\# Практическая работа №17: Интерактивный сплит-экран

Новицкий Кирилл ИСУ-24-2



\## Описание проекта

В данной работе реализован интерактивный сплит-экран с эффектом «шторки». При прокрутке страницы вниз (скролле) левая панель с текстом плавно сужается, полностью открывая фоновое изображение. После того как шторка исчезает, осуществляется переход на второй экран с полноэкранным изображением и наложенным поверх текстом.



\## Используемые технологии

\- \*\*HTML5\*\* — семантическая разметка структуры страницы.

\- \*\*CSS3\*\* — стилизация, использование Flexbox, `position: sticky` для фиксации экранов и плавные переходы.

\- \*\*JavaScript (Vanilla JS)\*\* — обработка события `scroll`, расчет текущей позиции прокрутки и динамическое изменение ширины левой панели.



\## Структура файлов

\- `index.html` — главная страница с разметкой.

\- `style.css` — таблица стилей.

\- `script.js` — скрипт, отвечающий за анимацию шторки.



\## Как запустить

1\. Склонируйте репозиторий или скачайте архив с проектом.

2\. Откройте файл `index.html` в любом современном веб-браузере (Chrome, Firefox, Safari и др.).

3\. Прокручивайте страницу вниз, чтобы увидеть анимацию.
